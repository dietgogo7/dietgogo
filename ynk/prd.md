# [PRD] YNK미디어 통합 웹 시스템 구축 정의서

## 1. 프로젝트 개요
* **프로젝트명:** YNK미디어 브랜드 사이트 및 통합 백오피스 구축
* **기본 컨셉:** '스킵과 로퍼'의 따뜻하고 산뜻한 감성을 담은 반응형 웹사이트
* **주요 목표:** * 브랜드 아이덴티티 전달 및 출판물(Y노블, Y코믹스 등) 홍보 효율화
    * 검색 엔진 최적화(SEO)를 통한 신규 독자 유입 극대화
    * 운영 효율성을 위한 PC 최적화 관리자 페이지 구축

## 2. 디자인 가이드라인 (Tone & Manner)
**'스킵과 로퍼' 모티브의 무해하고 산뜻한 디자인**

* **Color:** * Primary: 파스텔 옐로우 (#FFF9C4), 스카이 블루 (#E3F2FD)
    * Point: 코랄 핑크 (#FFEBEE) 및 차분한 그린 계열
    * Background: 웜 화이트 (#FFFAFA) 또는 연한 아이보리
* **UI 스타일:** * 모든 버튼과 카드 컴포넌트에 큰 `border-radius` 적용 (둥근 형태)
    * 선보다는 면과 그림자(Box-shadow)를 활용한 부드러운 구분
    * jQuery `animate()` 또는 CSS Transition을 활용한 페이드 인/아웃 효과 다수 활용
* **Typography:** 가독성이 좋은 둥근 고딕 계열 (예: 나눔스퀘어라운드 등)

## 4. 프론트엔드 - 사용자 페이지 (Responsive Web)

### 4.1 SPA 구조 구현 전략
* **Router:** `hashchange` 이벤트 또는 History API를 활용한 jQuery 기반 클라이언트 라우팅 구현.
* **Content Loading:** 상/하단 GNB와 Footer는 고정하고, 중앙 컨텐츠 영역만 `$.ajax`를 통해 동적으로 로드.
* **SEO 대응:** * 초기 진입 시 SSR(Server-Side Rendering) 방식으로 HTML을 내려주어 구글 봇이 컨텐츠를 수집할 수 있도록 구성.
    * 각 작품 상세 페이지별 동적 Meta Tag 및 JSON-LD(Structured Data) 삽입.

### 4.2 주요 메뉴 구성
1.  **Home:** 메인 비주얼 슬라이드, 신간 큐레이션, 레이블 바로가기.
2.  **About:** YNK미디어 비전, 연혁, 파트너사 정보.
3.  **Works:** * Y노블 / Y코믹스 레이블별 탭 구성.
    * 장르/발행일 기준 필터링 및 작품 검색 기능.
4.  **News & Event:** 공지사항, 이벤트 진행 현황 (카드 타입 리스트).
5.  **Contact:** * 원고 투고 폼 (파일 업로드 포함).
    * 제휴 및 일반 문의 폼.

## 5. 관리자 페이지 (PC 전용 Admin)

### 5.1 설계 원칙
* **Grid System:** 대량의 데이터를 한눈에 볼 수 있도록 데이터 그리드 중심의 레이아웃.
* **UX:** 폼 입력 시 유효성 검사(Validation) 즉시 피드백 제공.

### 5.2 주요 관리 기능
1.  **대시보드:** 방문 통계 차트, 신규 투고 현황, 미답변 문의 알림.
2.  **작품 관리:** * 작품 등록/수정/삭제 (표지 이미지, 시놉시스, ISBN, 태그 등).
    * 작품별 상세 페이지 SEO 키워드 관리.
3.  **전시 관리:** * 메인 페이지 롤링 배너 순서 및 노출 기간 설정.
    * '추천 작' 섹션 수동 큐레이션 관리.
4.  **고객 문의 관리:** * 원고 투고 파일 다운로드 및 검토 상태(접수/검토중/완료) 변경.
    * 제휴 문의 답변 작성 및 히스토리 관리.
5.  **게시판 관리:** WYSIWYG 에디터를 활용한 공지 및 이벤트 게시물 작성.
